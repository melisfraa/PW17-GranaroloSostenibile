## 📌 Descrizione del Progetto
Sviluppo di un'interfaccia web interattiva dedicata alla divulgazione del **Bilancio di Sostenibilità 2024 del Gruppo Granlatte-Granarolo**. Il progetto si focalizza sulla trasformazione di dati complessi e KPI aziendali in un'esperienza digitale moderna, accessibile e orientata all'utente.

L'elaborato approfondisce l'uso delle tecnologie web come strumento di **Responsabilità Sociale d'Impresa (CSR)**, facilitando la consultazione dei dati e il download dei report ufficiali.

## 🚀 Caratteristiche Tecniche
* **Interattività Pure CSS**: Utilizzo del *CSS Checkbox Hack* per la gestione di menu mobile e card espandibili, garantendo una UX fluida senza l'ausilio di JavaScript.
* **Architettura Mobile-First**: Layout completamente responsive ottimizzato per ogni dispositivo tramite Flexbox e CSS Grid.
* **Accessibilità**: Implementazione degli standard WCAG 2.1, uso di attributi ARIA e contrasti cromatici ottimizzati per screen reader.
* **Design Semantico**: Struttura HTML5 pulita per favorire l'indicizzazione (SEO) e la leggibilità del codice.

## 🛠️ Stack Tecnologico
* **HTML5**: Struttura semantica dell'informazione.
* **CSS3**: Styling avanzato, animazioni e gestione del layout.
* **Zero Framework**: Sviluppo realizzato interamente in codice nativo per massimizzare performance e sicurezza, seguendo le indicazioni del template di progetto
